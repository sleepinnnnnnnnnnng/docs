# 加入 NEC & 资料收集

欢迎加入 NEC 新能源开发者社区！请根据你的需求点击下方卡片或按下方步骤完成入群、资料登记与第一个贡献。

---

## 飞书资料收集

新人入群后，运营同学会在 QQ 群中发布飞书收集表二维码/链接，用于登记成员信息、学习意向与技能方向，方便后续分组与资源推送。

<CardGroup cols={2}>
  <Card title="飞书收集表 ①：成员登记" icon="clipboard">
    填写姓名、学校/公司、联系方式与感兴趣的技术方向，方便运营同学拉你入对应 SIG 群。
  </Card>
  <Card title="飞书收集表 ②：意向与技能" icon="clipboard-list">
    填写你的技能栈（机械 / 硬件 / 嵌入式 / 视觉 / 运营 / 项管）与可投入时间，便于项目分组。
  </Card>
</CardGroup>

<Tip>
  飞书收集表链接由运营同学在 QQ 群内定期更新。如果链接失效，请直接在 NEC QQ 群（479053780）中 @运营 或提交 [Gitee Issue](https://gitee.com/darrenpig/new_energy_coder_club/issues)。
</Tip>

加入 NEC QQ 交流群，与其他成员实时交流技术问题、竞赛动态与活动通知。

## 入群须知

1. **修改群昵称**：姓名 + 学校/公司 + 技术方向
2. **先读文档**：建议先浏览 [Start Here](/wiki/start-here) 与 [新人 60 分钟上手](/start-here/60min)
3. **规范提问**：描述清楚问题背景、已尝试方案与报错信息，避免只发"我这报错了怎么办"
4. **积极参与**：从 `good first issue` 开始，逐步参与真实项目
