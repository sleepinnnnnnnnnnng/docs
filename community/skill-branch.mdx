# SKill 分支文档

NEC 主仓库在 Gitee 上维护了一个 **`SKill` 分支**，用于沉淀与整理技能学习、知识分享、教程资料等内容。本页面作为该分支的文档入口，方便新人快速定位学习资源。

## 分支说明

- **分支地址**：[https://gitee.com/darrenpig/new_energy_coder_club/tree/SKill/](https://gitee.com/darrenpig/new_energy_coder_club/tree/SKill/)
- **用途**：技能教程、学习笔记、经验分享、工具链使用说明等非代码资产
- **维护方式**：社区成员可提交 PR，持续补充各领域技能资料

## 建议贡献内容

如果你有想要分享的技能资料，欢迎向 `SKill` 分支提交 PR：

- 机械设计教程（SolidWorks、3D 打印、加工工艺）
- 嵌入式开发笔记（STM32、ESP32、电机驱动）
- 算法学习路径（视觉、路径规划、控制算法）
- 工具链使用说明（Git、飞书、Gitee、Markdown）
- 竞赛经验复盘与踩坑记录

更多贡献规范请参考 [贡献指南](/contributing/guidelines)。
