## 安装 Mintlify

首先，确保安装了 Node.js (18+)，然后安装 Mintlify CLI：

```bash
npm install -g mintlify
```

## 本地预览

在项目根目录运行：

```bash
mint dev
```

本地服务器将在 `http://localhost:3000` 启动，自动检测文件变化并热更新。

## 创建新页面

1. 创建 `.mdx` 文件
2. 添加 frontmatter：
```yaml
---
title: "页面标题"
description: "页面描述"
---
```
3. 在 `docs.json` 中添加导航

## 组件使用

### 卡片

```mdx
<Card title="标题" icon="icon-name">
  内容描述
</Card>
```

### 卡片组

```mdx
<CardGroup cols={2}>
  <Card title="卡片1">内容1</Card>
  <Card title="卡片2">内容2</Card>
</CardGroup>
```

### 步骤

```mdx
<Steps>
  <Step title="步骤1">说明1</Step>
  <Step title="步骤2">说明2</Step>
</Steps>
```

## 检查链接

运行以下命令检查死链：

```bash
mint broken-links
```

## 部署

文档会自动部署到 Mintlify 平台。提交更改到仓库后：

1. 连接 Gitee/GitHub 仓库到 Mintlify
2. 推送更改到主分支
3. 自动构建和部署
