## 三步加入 NEC

<Steps>
  <Step title="加入社区" icon="user-plus">
    扫码进入飞书/微信群，获取仓库权限与赛季通知。

    <Card
      title="加入社区"
      icon="message"
      href="/community/join-collection"
      horizontal
    >
      查看入群二维码
    </Card>
  </Step>

  <Step title="环境搭建" icon="laptop">
    克隆 NEC 主仓库，验证本地开发环境。

    ```bash
    git clone https://gitee.com/darrenpig/new_energy_coder_club.git
    cd new_energy_coder_club
    python --version
    ```
  </Step>

  <Step title="提交第一个 PR" icon="git-pull-request">
    跟着新人 onboarding 跑通 Demo，完成首次贡献。

    <Card
      title="CURC 2027 新人入队"
      icon="play"
      href="/start-here/curc2027-onboarding"
      horizontal
    >
      跑通 Demo → 提交 PR
    </Card>
  </Step>
</Steps>
