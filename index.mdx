## 三步加入 NEC

<Steps>
  <Step title="加入社区" icon="user-plus">
    扫码进入飞书/微信群，获取仓库权限、培训资料与赛季通知。

    <Card
      title="加入社区"
      icon="message"
      href="/community/join-collection"
      horizontal
    >
      查看入群二维码与资料收集入口
    </Card>
  </Step>

  <Step title="环境搭建" icon="laptop">
    安装 Git 与 Python 3.8+，克隆 NEC 主仓库，验证本地环境。

    ```bash
    git clone https://gitee.com/darrenpig/new_energy_coder_club.git
    cd new_energy_coder_club
    python --version
    ```
  </Step>

  <Step title="提交第一个 PR" icon="git-pull-request">
    跟着 60 分钟快速入门，跑通最小 Demo，完成新人墙登记，发起你的第一个 Pull Request。

    <Card
      title="60 分钟快速入门"
      icon="play"
      href="/start-here/60min"
      horizontal
    >
      跑通 Demo → 提交 PR → 领取 good first issue
    </Card>
  </Step>
</Steps>

## SKill 技能分支

NEC 主仓库维护了一个 **`SKill` 分支**，用于沉淀技能教程、学习笔记、经验分享、工具链使用说明等非代码资产。新人可以在这里找到机械、嵌入式、算法、工具链等领域的学习资料。
