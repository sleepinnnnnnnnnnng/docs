### 每个 Skill 学什么

<CardGroup cols={2}>
  <Card title="机械设计" icon="cube">
    **适合**：想参与机器人结构设计、加工装配的同学

    **包含内容**：
    - SolidWorks / Fusion 360 三维建模
    - 3D 打印切片与后处理
    - CNC、线切割、钣金等加工工艺
    - 标准件选型（轴承、电机、丝杠等）

    **推荐起点**：[SKill/mechanical](https://gitee.com/darrenpig/new_energy_coder_club/tree/SKill/mechanical/)
  </Card>

  <Card title="嵌入式开发" icon="microchip">
    **适合**：想写单片机、驱动电机、调传感器的同学

    **包含内容**：
    - STM32 / ESP32 开发环境配置
    - CAN、PWM、FOC 电机驱动
    - IMU、摄像头、激光雷达等传感器使用
    - FreeRTOS / RT-Thread 实时控制

    **推荐起点**：[SKill/embedded](https://gitee.com/darrenpig/new_energy_coder_club/tree/SKill/embedded/)
  </Card>

  <Card title="算法" icon="brain">
    **适合**：对视觉、规划、控制、AI 感兴趣的同学

    **包含内容**：
    - OpenCV / YOLO 视觉识别
    - 路径规划与运动控制
    - SLAM 入门与建图定位
    - 数据集标注与模型训练

    **推荐起点**：[SKill/algorithms](https://gitee.com/darrenpig/new_energy_coder_club/tree/SKill/algorithms/)
  </Card>

  <Card title="软件工程" icon="code">
    **适合**：想规范写代码、做工具链、搞上层软件的同学

    **包含内容**：
    - Git 工作流与 Conventional Commits
    - Python 工程化与脚本自动化
    - ROS2 节点开发与调试
    - 代码规范、CI/CD、文档工程

    **推荐起点**：[SKill/software](https://gitee.com/darrenpig/new_energy_coder_club/tree/SKill/software/)
  </Card>

  <Card title="工具链" icon="wrench">
    **适合**：所有新人，先学会社区协作工具

    **包含内容**：
    - 飞书文档、多维表格、妙记使用
    - Gitee 仓库操作与 PR 流程
    - Markdown / MDX 文档写作
    - Windows / Linux / WSL 开发环境配置

    **推荐起点**：[SKill/tools](https://gitee.com/darrenpig/new_energy_coder_club/tree/SKill/tools/)
  </Card>

  <Card title="竞赛经验" icon="trophy">
    **适合**：准备参加 ROBOCON、RM、节能赛等比赛的同学

    **包含内容**：
    - 赛季复盘与失败案例分析
    - 规则解读与策略拆解
    - 备赛时间线与任务拆分模板
    - 预算申报、采购流程、场地调试经验

    **推荐起点**：[SKill/competition](https://gitee.com/darrenpig/new_energy_coder_club/tree/SKill/competition/)
  </Card>
</CardGroup>

## 三步加入 NEC

<Steps>
  <Step title="加入社区" icon="user-plus">
    扫码进入飞书/微信群，获取仓库权限、培训资料与赛季通知。

    <Card
      title="加入社区"
      icon="message"
      href="/community/join-collection"
      horizontal
    >
      查看入群二维码与资料收集入口
    </Card>
  </Step>

  <Step title="环境搭建" icon="laptop">
    安装 Git 与 Python 3.8+，克隆 NEC 主仓库，验证本地环境。

    ```bash
    git clone https://gitee.com/darrenpig/new_energy_coder_club.git
    cd new_energy_coder_club
    python --version
    ```
  </Step>

  <Step title="提交第一个 PR" icon="git-pull-request">
    跟着 60 分钟快速入门，跑通最小 Demo，完成新人墙登记，发起你的第一个 Pull Request。

    <Card
      title="60 分钟快速入门"
      icon="play"
      href="/start-here/60min"
      horizontal
    >
      跑通 Demo → 提交 PR → 领取 good first issue
    </Card>
  </Step>
</Steps>
