# 主轴雕刻机软件安装指南

本文档整理 NEC A416 实验室主轴雕刻机（CNC V4 控制卡）所需的软件安装流程，包括 USB 驱动、控制软件、NC 转换器与精雕软件。完成安装后，可继续阅读 [刀路生成](/mechanical/cnc-engraver-toolpath) 与 [软件端操作](/mechanical/cnc-engraver-operation) 文档开始加工。

<Warning>
  安装驱动与控制软件前，请勿提前将雕刻机通过 USB 连接到电脑，建议按本文档顺序操作，避免 Windows 设备管理器出现未知设备。
</Warning>

## 安装包目录说明

配套资料位于仓库 `雕刻机资料V3/` 目录：

| 目录 | 内容 |
|---|---|
| `1控制软件和驱动程序/` | 雕刻机 USB 驱动、CNC V4 控制软件压缩包 |
| `2精雕和NC转换器/` | 精雕软件安装包、NC 转换器、帮助资料与视频教程 |
| `演示素材/` | 示例位图、DXF 素材与测试刀路 |

原始 PDF、驱动与软件安装包可在 [主轴雕刻机原始资料下载](/mechanical/cnc-engraver-resources) 页面获取。

## 安装 USB 驱动

1. 在 `1控制软件和驱动程序/` 中找到雕刻机驱动安装程序。
2. 双击运行，按向导提示完成安装。
3. 安装完成后，再使用 USB 线将雕刻机与电脑连接。
4. 打开 Windows 设备管理器，确认出现正确的 COM 端口（常见为 `COM3`）。

<Note>
  若设备管理器中出现带感叹号的未知设备，请重新插拔 USB 线或手动指定驱动路径为 `1控制软件和驱动程序/` 目录。
</Note>

## 安装控制软件

1. 将 `1控制软件和驱动程序/2控制软件.rar` 解压到常用目录，例如 `D:\CNC\CNCV4\`。
2. 解压完成后，目录中应包含类似 `CNCV4.exe` 的可执行文件。
3. 双击 `CNCV4.exe` 即可运行，无需额外安装。

首次启动时，若提示选择串口，请参考 [软件端操作流程](/mechanical/cnc-engraver-operation#连接设备) 进行连接。

## 安装 NC 转换器

NC 转换器负责将精雕软件生成的 `.eng` 刀路文件转换为雕刻机可直接识别的 `.nc` 格式。

1. 打开 `2精雕和NC转换器/` 目录。
2. 将 NC 转换器相关文件复制到电脑硬盘（建议与控制软件放在同一父目录下，便于管理）。
3. 若转换器为绿色软件，解压后直接运行主程序即可。

<Info>
  精雕生成的 `.eng` 文件不能直接载入 CNC V4 控制软件，必须经过 NC 转换器转换为 `.nc` 格式后再使用。
</Info>

## 安装精雕软件

精雕软件用于绘制矢量图形并生成加工刀路。

1. 进入 `2精雕和NC转换器/1精雕软件/`。
2. 双击 `setup.exe` 开始安装。
3. 安装过程中需要输入安装密码：

   ```text
   JA2CNC01
   ```

4. 按向导完成安装，建议保持默认安装路径以避免路径中包含中文或特殊字符。
5. 安装完成后，可参考目录中的视频教程学习基本操作。

<Tip>
  精雕软件功能较多，本站点仅整理与雕刻机配套的核心流程。复杂工艺参数请参考 `2精雕和NC转换器/` 中的帮助资料或自行查阅网络教程。
</Tip>

## 连接与验证

完成全部软件安装后，按以下顺序验证：

<Steps>
  <Step title="连接 USB">
    确保雕刻机电源关闭，连接 USB 数据线，再打开雕刻机电源。
  </Step>
  <Step title="确认 COM 口">
    在 Windows 设备管理器中查看 COM 口号，与控制软件中选择的一致。
  </Step>
  <Step title="打开控制软件">
    运行 `CNCV4.exe`，进入“设置”页面选择对应 COM 口与波特率（通常为 115200）。
  </Step>
  <Step title="点击连接">
    点击“连接机器”，确认状态由“未连接”变为已连接。
  </Step>
</Steps>

## 常见问题

<AccordionGroup>
  <Accordion title="设备管理器看不到 COM 口？">
    1. 更换 USB 接口（优先使用 USB 2.0 或主板直连口）；
    2. 重新安装驱动；
    3. 检查 USB 线是否仅为充电线（需为数据线）。
  </Accordion>
  <Accordion title="控制软件提示连接失败？">
    1. 确认 COM 口与波特率正确；
    2. 关闭其他占用串口的软件；
    3. 点击软件“重启”按钮重新枚举串口。
  </Accordion>
  <Accordion title="精雕安装时提示密码错误？">
    安装密码区分大小写，请准确输入 `JA2CNC01`。若仍失败，请检查安装包是否完整。
  </Accordion>
</AccordionGroup>
