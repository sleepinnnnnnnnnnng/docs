# 主轴雕刻机软件端操作流程

本文档整理了 NEC A416 实验室主轴雕刻机（CNC V4 控制软件）的软件端操作流程，涵盖设备连接、任务载入、XY 定位、Z 轴对刀与启动加工等关键步骤，供新人快速上手与现场备查。

如果你是第一次使用这台雕刻机，建议按以下顺序阅读：

1. [软件安装指南](/mechanical/cnc-engraver-installation) — 安装驱动、控制软件、精雕与 NC 转换器
2. [刀路生成与 NC 转换](/mechanical/cnc-engraver-toolpath) — 使用精雕出刀路并转换为 `.nc` 文件
3. 本文档 — 在控制软件中完成对刀与加工

<Warning>
  操作前请确认已佩戴护目镜，长发需束起，禁止戴手套靠近旋转主轴。加工过程中必须有人值守。
</Warning>

## 前置准备

- 确认雕刻机已接通电源并处于待机状态
- 确认电脑已安装 CNC V4 控制软件
- 准备好待加工的 `.nc` 格式刀路文件
- 检查刀具是否安装牢固、玻纤板是否固定可靠
- 确认当前刀头位置为工件零点后再载入任务文件（控制软件默认以当前位置为加工原点）

## 软件界面概览

控制软件主要分为左侧导航栏与右侧可视化区：

![控制软件主界面](/images/cnc-engraver/control-panel.png)

左侧导航栏从上到下依次为：**任务、控制、控制 Ⅱ、命令、设置、提示、开始、重启**。右侧显示当前刀路预览、坐标与机器状态。

## 操作步骤

<Steps>
  <Step title="连接设备" icon="plug">
    1. 插上数据线，将电脑与雕刻机连接。
    2. 点击左侧"设置"，进入串口设置页面。
    3. 选择正确的 COM 口（常见为 COM3）与波特率（通常为 115200）。
    4. 点击"连接机器"，确认顶部"机器状态"由"未连接"变为已连接。

    ![串口设置与连接机器](/images/cnc-engraver/settings-serial.png)

    若无法识别设备，可尝试更换 USB COM 口，或点击"重启"重新枚举串口。
  </Step>

  <Step title="载入加工任务" icon="file-arrow-up">
    1. 点击左侧"任务"，进入任务管理页面。
    2. 点击"载入任务"，选择 `.nc` 格式的刀路文件。

    ![任务页面载入任务](/images/cnc-engraver/task-load.png)

    载入后，右侧会显示刀路轮廓预览，同时可看到文件行数、预计加工时间、工作零点等信息。

    ![已载入任务的刀路预览](/images/cnc-engraver/task-preview.png)
  </Step>

  <Step title="校准 X/Y 轴位置" icon="arrows-up-down-left-right">
    1. 点击左侧"控制"，进入运动控制页面。
    2. 以刀路路径轮廓数据为依据，使用 **X+ / X- / Y+ / Y-** 按钮移动刀头。
    3. 确保刀头运作路径位于玻纤板的正确区域内，避免加工到夹具或台面。

    ![运动控制与功能控制](/images/cnc-engraver/control-functions.png)
  </Step>

  <Step title="Z 轴对刀" icon="arrow-down">
    对刀是加工前最关键的步骤，请严格按照以下顺序操作：

    1. 点击左侧"控制 Ⅱ"，将移动单位调整为 **0.1 mm**，移动速度建议保持默认值。

       ![移动单位与速度设置](/images/cnc-engraver/movement-unit.png)

    2. 返回"控制"页面，利用方向按钮缓慢放下刀头。
    3. 在刀头快要贴近玻纤板时（肉眼观测约 1–2 mm），停止快速移动。
    4. 点击"主轴"按钮，启动刀头。
    5. 切换到"控制 Ⅱ"页面，确认移动单位为 **0.1 mm**。
    6. 点按"Z-"，使刀头缓慢接近玻纤板。
    7. 持续下刀，最终进刀约 **0.3–0.6 mm**。

    ![控制 II 页面精调参数](/images/cnc-engraver/control-ii.png)

    <Tip>
      对刀时建议先手动转动主轴确认刀具无摆动，再启动主轴，防止刀具偏心刮伤板材。
    </Tip>
  </Step>

  <Step title="开始加工" icon="play">
    1. 确认刀路、坐标与下刀深度均无误。
    2. 检查"进给速率"是否为预期值（默认 100%）。
    3. 点击左侧"开始"按钮启动加工。

    ![开始加工前的控制页面](/images/cnc-engraver/control-preview.png)

    加工过程中保持观察，发现异常立即点击"暂停"或"急停"。
  </Step>
</Steps>

## 控制面板常用按钮

| 按钮/功能 | 作用说明 |
|---|---|
| 主轴 | 启动/停止主轴电机 |
| Z+ / Z- | 控制刀头上下移动 |
| X+ / X- | 控制刀头左右移动 |
| Y+ / Y- | 控制刀头前后移动 |
| 对刀 | 记录当前 Z 轴为加工零点 |
| 归位 | 让各轴回到机械原点 |
| 解除锁定 | 解除电机锁定，可手动移动轴 |
| 移动单位 | 精调时建议设置为 0.1 mm |
| 移动速度 | 手动移动时的速度，数值越小越慢 |
| 开始/暂停/急停 | 控制加工启停，急停用于突发情况 |

## 手动控制主轴

控制软件界面上没有专门的“开启主轴”按钮。如需手动启动主轴（例如对刀前检查刀具），可在命令行控制中输入以下 G 代码：

```text
M3 S24000
```

其中 `M3` 表示主轴正转，`S24000` 表示转速 24000 RPM。

关闭主轴命令：

```text
M5
```

<Tip>
  手动转动 Z 轴联轴器可调节刀头高度，但请在断电或解除电机锁定后操作，避免损坏驱动器。
</Tip>

## 安全注意事项

- 主轴旋转时严禁用手触摸刀具或清理碎屑。
- 加工玻纤板会产生粉尘，建议开启吸尘器并佩戴口罩。
- 对刀完成后，务必再次确认 Z 轴零点与下刀深度。
- 首次使用或更换刀具后，建议先用废料试切，验证刀路无误。
- 急停按钮仅在紧急情况下使用，按下后需按说明书复位。

## 常见问题

<AccordionGroup>
  <Accordion title="软件无法连接雕刻机怎么办？">
    1. 检查数据线是否插紧；
    2. 在设备管理器中确认 COM 口号；
    3. 在软件中手动选择正确的 COM 口；
    4. 点击软件"重启"按钮重新连接。
  </Accordion>
  <Accordion title="刀路预览与实际位置偏差怎么办？">
    1. 检查 `.nc` 文件坐标原点是否与板材装夹原点一致；
    2. 重新校准 X/Y 轴起点；
    3. 确认没有误触"偏移"或"镜像"选项。
  </Accordion>
  <Accordion title="下刀深度应该设置多少？">
    玻纤板常规雕刻进刀约 0.3–0.6 mm。具体深度需根据板材厚度、刀具直径与主轴转速微调，首次加工建议从较小深度开始试切。
  </Accordion>
  <Accordion title="自动开启主轴需要勾选吗？">
    建议在正式加工时勾选"自动开启主轴"，避免手动操作遗漏；对刀阶段可关闭，便于手动控制。
  </Accordion>
  <Accordion title="加工原点应该在哪里设置？">
    控制软件载入任务后，默认将当前刀头位置作为工件零点。因此建议先移动刀头到目标原点，再载入 `.nc` 文件。
  </Accordion>
  <Accordion title="主轴无法手动启动？">
    控制软件界面未提供主轴启动按钮。如需手动启动，请在命令行中输入 `M3 S24000`；关闭则输入 `M5`。
  </Accordion>
</AccordionGroup>
