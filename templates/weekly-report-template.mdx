# 周报模板

> 本文件为模板，不直接展示在站点导航中。发布新一期周报时，请复制本文件到 `community/weekly/YYYY-MM-DD.mdx` 并填充实际内容。

## 基本信息

- **周报期数**：第 N 期
- **时间范围**：YYYY-MM-DD ~ YYYY-MM-DD
- **编辑**：@姓名
- **审核**：@姓名

## 本周精选

### 1. 新增贡献

- PR / 文档 / 教程链接
- 贡献者：@姓名
- 一句话价值总结

### 2. 新手任务

- 本周推荐的 3 个 good first issue
- 适合方向：机械 / 硬件 / 嵌入式 / 视觉 / 运营

### 3. 项目进展

| 项目 | 本周里程碑 | 负责人 |
|------|-----------|--------|
| 项目名称 | 完成内容 | @姓名 |

### 4. 踩坑复盘

- 问题描述
- 原因分析
- 解决方案 / 可复用经验

### 5. 下周计划

- 重点任务
- 招募需求（机械 / 硬件 / 软件 / 运营）
- 会议或活动时间

### 6. NEC+ 会员内容预告

- Office Hour 主题
- 新增模板或资料
- 会员专属活动
