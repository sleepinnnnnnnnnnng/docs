**关键决策记录**：

- [x] **机械方案选型依据**：以 SolidWorks + Rhino 为主工作流，优先采用参数化建模与可加工性验证，降低 3D 打印与机加工返工率。
- [x] **电控板 DM-MC02 选型原因**：STM32H723VG 性能满足多电机 CAN 总线通信需求，配套达妙生态文档完整，社区开源例程丰富。
- [x] **视觉方案 K230 → Odin1 切换原因**：K230 用于基础视觉识别验证，Odin1 在赛场导航与实时定位场景下算力与接口更匹配。
- [x] **舵轮 T30/U8 调试经验**：优先完成单轮速度闭环与转向零点标定，再接入整车运动学模型，避免多轮耦合调试难度叠加。

**踩坑汇总**：

- [x] **电机驱动接线问题**：宇树 8010 电机红黑线顺序与部分电调默认定义相反，上电前务必对照官方接线图用万用表确认相序。
- [x] **电池规格不匹配**：整车大功率电机场景需使用 6S 锂聚合物电池，临时用 3S/4S 电池会导致瞬时掉压与驱动保护。
- [x] **气动夹爪元件选型**：需同时考虑气源压力、电磁阀响应速度与夹爪行程，建议先做单轴抓取力测试再集成到机械臂末端。
